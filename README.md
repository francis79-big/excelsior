This repository containes Excelsior hotel's website data.

Used languages and technologies: 
  - HTML
  - CSS
  - Java Script
  - PHP
  - SQL database
  - Xampp
    
The website has a few pages, for example: galeria and booking possibility with registration fields.
The visitor only can makes booking after a registration.

Also available in this repository an sql file. It was created via exporting process from the Xampp PHPMyAdmin. 
It contains the necessary tables like the room types and there's equipments or the registered users and there's reservations.
